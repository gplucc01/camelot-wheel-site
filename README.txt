Put your image files in THIS folder, named exactly:

  icon.png         - your app icon (square, 1024x1024 is fine)
  screenshot1.png  - App Store screenshot 1
  screenshot2.png  - App Store screenshot 2
  screenshot3.png  - App Store screenshot 3
  screenshot4.png  - App Store screenshot 4

Lowercase names, .png extension. GitHub Pages is case-sensitive.
If your files are .jpg, either rename them to .png-style names above
won't work - instead edit index.html and change the src= extensions to .jpg.

You can delete this README.txt after adding your images.
